# Prompt

This repository contains the prompt file that Magna uses to guide LLMs for vibe coding. It serves as a lightweight, human-readable reference for how prompts, rules, and tasks are organized in this project.

## Repository Contents

- **README.md**: Overview of the repository and the purpose of each file.
- **History.md**: Reserved for capturing prompt evolution, revisions, or change history.
- **Rules.md**: Intended to document the rules or constraints used to shape prompts and LLM behavior.
- **Struture.md**: A placeholder for describing prompt structure, formatting, or organization patterns.
- **Task.md**: A space for listing tasks or checklists related to prompt updates and usage.
